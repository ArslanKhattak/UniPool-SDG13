# UniPool-SDG13

# Project Title:
# UniPool – Shared Ride for Climate Action

# SDG Chosen:
**SDG 13 – Climate Action**
Our app prototype supports Sustainable Development Goal 13 by reducing fuel consumption and carbon emissions through shared transportation among the university students and the faculty.

# Project Description
UniPool is a ride sharing mobile application buit for university stundents and its faculty that allows multiple users traveling to the same university to share one ride.  
The system reduces fuel usage, helps the users by offering them lower transportation costs, and minimizes carbon emissions. 

Target Users:
- University Students
- University Teachers 
- University Faculty
